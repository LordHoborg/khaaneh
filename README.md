# خانه (Khaaneh) — Smart Persian Kitchen Assistant

A production-ready, modern, mobile-first Persian cooking assistant built with Next.js.

## Features Implemented
- Hero landing with prominent "امروز چه بپزم؟" CTA
- Smart Pantry with live expiry alerts and add/delete
- AI-powered Recommendations (ranked top meals with transparent reasons)
- Weekly Meal Planner
- Detailed Recipe pages with cultural notes + AI chat
- Profile & Settings

## Tech Stack
Next.js 15 + TypeScript + Tailwind + shadcn-inspired UI + Prisma

## Run Locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

The full architecture document is available in /docs

Ready for Supabase integration, OpenAI, and production deployment.
